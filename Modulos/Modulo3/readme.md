
# Análise de Desempenho Escolar 📊

Este projeto consiste numa análise exploratória de dados (EDA) focada no desempenho académico de alunos, utilizando Python e as bibliotecas **Matplotlib** e **Seaborn** para visualização de dados.

## 📁 Organização do Projeto

O projeto está estruturado da seguinte forma:

* **`dataset_desempenho_escolar.csv`**: Base de dados contendo informações sobre os alunos (Nomes, Notas e Frequência).
* **`notebook_visualizacao_dados_com_5_questoes.ipynb`**: Notebook didático com introdução à visualização e lista de exercícios.
* **`gabarito_analise_desempenho.ipynb`**: Resolução completa e visual das questões propostas.

## 🚀 Instruções de Execução

### Pré-requisitos
Para rodar este projeto, necessita de ter o Python instalado com as seguintes bibliotecas:
* Pandas
* Matplotlib
* Seaborn

Pode instalá-las via terminal:

pip install pandas matplotlib seaborn
